# Track future Apple Health export metrics roadmap

Track future export-metric expansion using the roadmap doc instead of keeping a long backlog directly in the issue.

Roadmap:

- [`docs/roadmap/future-export-metrics.md`](docs/roadmap/future-export-metrics.md)

Scope of the roadmap includes:

- Cardio and recovery metrics
- Sleep metrics
- Activity and movement metrics
- Body composition metrics
- Mobility and gait metrics
- Workout summaries
- Cycling and performance metrics
- Hearing and environmental exposure metrics
- Nutrition
- Clinical-record and file-backed exports such as ECGs and workout routes

Follow-up implementation work should be split into smaller issues by theme.
